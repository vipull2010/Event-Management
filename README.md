# Event-Management

Event Management System (Frontend)
A sleek, responsive, and modern landing page for an Event Management System. This project showcases the ability to create a professional user interface for managing conferences, weddings, workshops, and corporate events using purely frontend technologies.

Features
Responsive Landing Page: Fully compatible with mobile, tablet, and desktop screens.

Event Categories: Beautifully designed cards for different event types.

Booking Form: A stylized user registration and event booking interface.

Navigation: Smooth scrolling navigation bar with hover effects.

Interactive Gallery: A section to showcase past successful events.

Contact Section: Integrated footer with social media links and contact details.

Technologies Used
HTML5: Used for semantic layout and structure.

CSS3: Used for custom styling, Flexbox/Grid layouts, and animations.

Google Fonts: Integrated for clean, modern typography.

Font Awesome: Utilized for high-quality icons.
